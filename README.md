# Meru Marketplace - E-Commerce Platform

A modern, fully-featured e-commerce platform built with HTML, CSS (Tailwind), and JavaScript. Buy and sell products locally in Meru County, Kenya.

## 🎯 Features

### For Buyers
- 🛍️ **Browse Products** - Explore thousands of items from local sellers
- 🛒 **Shopping Cart** - Easy-to-use cart system
- ⭐ **Ratings & Reviews** - See what other customers think
- 🔐 **Secure Checkout** - Safe payment processing
- 📱 **Mobile Responsive** - Shop on any device

### For Sellers
- 🏪 **Store Setup** - Create your own store in minutes
- 📦 **Product Listing** - Easy product management
- 📊 **Analytics** - Track your sales and performance
- 💳 **Payment Processing** - Secure transactions
- 📈 **Sales Growth** - Tools to expand your business

### Authentication
- 👤 **User Login** - Secure account access
- ✍️ **Sign Up** - Create new accounts
- 🔑 **Password Protection** - Encrypted credentials
- 📱 **Phone Verification** - Additional security

## 🛠️ Technology Stack

- **Frontend**: HTML5, Tailwind CSS, JavaScript
- **Icons**: Font Awesome 6.4.0
- **Responsive**: Mobile-first design
- **Performance**: Optimized animations and transitions

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Yvonnne16025/Research.git
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - No installation required!

3. **Explore the marketplace**
   - Browse featured products
   - Try the login/signup system
   - Check out seller information

## 📋 How to Use

### For Buyers:
1. Click "Start Shopping" on the homepage
2. Browse available products
3. Click "Add to Cart" on items you like
4. Click "Login" to access your account
5. Proceed to checkout

### For Sellers:
1. Click "Start Selling" or "Create Seller Account"
2. Fill in your business information
3. Start listing your products
4. Manage orders and track sales
5. Grow your business!
